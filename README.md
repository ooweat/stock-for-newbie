# Stock For Newbie

국내주식 및 미국주식 증권사 수수료를 비교할 수 있는 계산기 서비스입니다.

실제 거래 시 발생하는:

- 증권사 거래 수수료
- 이벤트 우대 수수료
- 환전 수수료
- 증권거래세

등을 계산하여 예상 비용과 실제 수익을 비교할 수 있습니다.

---

## Features

### 국내 / 해외 주식 지원

- 국내주식 수수료 비교
- 미국주식 수수료 비교
- 환전 수수료 계산
- 증권거래세 계산

### 실시간 환율

- USD/KRW 환율 자동 조회
- 5분 캐시 적용
- 환율 기준 시간 표시

### 증권사 비교

- 총 비용 기준 정렬
- 증권사 수수료만 비교 가능
- 이벤트 수수료 표시
- 환율 우대 표시

### UX

- 모바일 최적화
- 실시간 계산
- 아코디언 상세 보기
- 빠른 금액 입력
